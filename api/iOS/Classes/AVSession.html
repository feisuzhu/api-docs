– sendMessage:transient:

发送消息

- (void)sendMessage:(AVMessage *)message transient:(BOOL)transient

Parameters

message

消息对象
transient

设置为 YES, 当且仅当某个 peer 在线才会收到该条消息，且该条消息既不会存为离线消息，也不会通过消息推送系统发出去. 如果设置为 NO, 则该条消息会设法通过各种途径发到 peer 客户端，比如即时通信、推送、离线消息等。

发送消息
