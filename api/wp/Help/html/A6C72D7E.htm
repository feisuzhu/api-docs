Query
Constructs a AVQueryT for AVInstallations. AVInstallation queries cannot be executed on the client, but are used to target push notifications.
TimeZone
The time zone in which this device resides. This string is in the tz database format AVOSCloud uses for local-time pushes. Due to platform restrictions, the mapping is less granular on Windows than it may be on other systems. E.g. The zones America/Vancouver America/Dawson America/Whitehorse, America/Tijuana, PST8PDT, and America/Los_Angeles are all reported as America/Los_Angeles.
UpdatedAt
Gets the last time this object was updated as the server sees it, so that if you make changes to a AVObject, then wait a while, and then call SaveAsync, the updated time will be the time of the SaveAsync call rather than the time the object was changed locally.
(Inherited from AVObject.)
